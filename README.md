boxen
=====

# Inital Setup

## Install xcode command line tool

```sh
$ xcode-select --install
```

## Create directory and clone this repo

```sh
$ mkdir src/
$ cd src/
$ git clone https://github.com/take/boxen.git
```

# Installation

```
$ cd /path/to/boxen
$ script/boxen --no-fde
```

`--no-fde` will disable hard drive encrypt
